# Проверка портов: nginx и backend

## Архитектура:
- **Backend (Go)**: работает на порту **8080**
- **nginx**: должен слушать на порту **80** и проксировать на **8080**

## Быстрая проверка на сервере:

```bash
# 1. Проверить конфигурацию nginx
sudo ./scripts/check-nginx-config.sh

# 2. Проверить, что backend работает на 8080
curl http://localhost:8080/health

# 3. Проверить, что nginx проксирует с 80 на 8080
curl http://localhost/health

# 4. Проверить порты (если есть ss)
sudo ss -tlnp | grep -E "80|8080"

# 5. Или через lsof
sudo lsof -i :80
sudo lsof -i :8080
```

## Ожидаемый результат:

- Backend на 8080: ✅ отвечает
- nginx на 80: ✅ проксирует на 8080
- Внешний доступ: ✅ работает (уже проверено)

Если все работает, можно запускать Certbot - он просто добавит HTTPS к существующей конфигурации.
